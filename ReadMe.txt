1- Run Backend with "java -jar insurance-0.0.1-SNAPSHOT.jar" inside this folder.
2- Run Frontend with "serve -s build" in "/ins-frontend" folder.

NOTE:If "serve" not recognized run "npm install -g serve" in terminal/command prompt. 
You can also run frontend in this folder with "npm install" then "npm start".

Used Java 8, Spring Boot 2, ReactJs.
